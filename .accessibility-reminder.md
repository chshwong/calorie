# Accessibility Reminder

## Standards
- **WCAG 2.0 AA** ✅
- **AODA 2.0 AA** ✅ (same as WCAG 2.0 AA)

## Always Include
- ✅ Accessibility labels for all interactive elements
- ✅ Proper form labels (not just placeholders)
- ✅ Minimum 44x44pt touch targets
- ✅ Color contrast ratios (4.5:1 normal, 3:1 large text)
- ✅ Keyboard navigation support
- ✅ Focus indicators
- ✅ Error announcements
- ✅ Screen reader support

## When Adding New Features
1. Use utilities from `utils/accessibility.ts`
2. Test with screen readers
3. Verify keyboard navigation
4. Check color contrast
5. Ensure touch target sizes

## Quick Reference
- Buttons: `getButtonAccessibilityProps()`
- Links: `getLinkAccessibilityProps()`
- Inputs: `getInputAccessibilityProps()`
- Icons: `decorative={true}` or `accessibilityLabel="..."`
- Touch targets: `getMinTouchTargetStyle()`
- Focus: `getFocusStyle(colors.tint)`
